Centos7安装Git和Maven

恶作剧 linuxgitmaven

安装环境：

Centos7

git-2.29.2

mven-3.6.1

安装git有两种方法，yum安装的版本比较旧

# 一、yum安装git

输入命令即可完成安装

yum -y install git
1

查看git版本

git --version
1

# 二、GitHub下载指定版本

  1. 进入git在GitHub上发布版本页面https://github.com/git/git/releases，这个页面我们可以找到所有git已发布的版本。这里我们选择最新版的tar.gz包

  2. 解压压缩包 tar -zxvf git-2.29.2.tar.gz

  3. 进入到解压后的文件夹

    cd git-2.29.2
    1

  4. 拿到解压后的源码以后我们需要编译源码了，不过在此之前需要安装编译所需要的依赖。

    yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc perl-ExtUtils-MakeMaker
    1
    2

    中途出现提示时，按y

  5. 安装编译源码所需依赖的时候，yum自动安装了git，这时候你需要先卸载这个旧版的git。

    yum -y remove git
    1

  6. 编译git源码

    make prefix=/usr/local/git all
    1

  7. 安装git至/usr/local/git路径

    make prefix=/usr/local/git install
    1

  8. 配置环境变量

    vi /etc/profile
    1

    底部添加

    export PATH=$PATH:/usr/local/git/bin
    1

  9. 刷新环境变量

    source /etc/profile
    1

  10. 查看Git是否安装成功

    git --version
    1

# 三、安装maven

Binary表示已经编译好的(推荐)

Source表示未编译的

  1. 官网下载maven版本

    下载地址：http://maven.apache.org/download.cgi

    各个版本下载：https://archive.apache.org/dist/maven/maven-3/

  2. 上传到linux的/usr/local目录,创建maven目录

  3. 解压文件到/usr/local/maven/下

    tar -zxvf apache-maven-3.6.1-bin.tar.gz
    1

  4. 配置环境变量

    vi /etc/profile
    1

    底部输入

    export PATH=$PATH:/usr/local/maven/maven-3.6.1/bin
    1

  5. 刷新环境变量

    source /etc/profile
    1

  6. 查看版本

    mvn -v
    1

注意：安装好maven后修改 配置文件/usr/local/maven/maven-3.6.1/conf/setting.xml 指定jar包的仓库地址

  1. 配置仓库地址

    <localRepository>/usr/local/maven/repo</localRepository>
    1
